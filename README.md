# 📚 E-Commerce Data Scraper & Analyst

This project is a high-performance web scraping tool built with Python to extract and analyze data from online bookstores. It demonstrates the ability to navigate complex HTML structures, handle pagination, and process raw data into structured formats.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Libraries:** BeautifulSoup4, Requests, Pandas
- **Environment:** Google Colab / VS Code

## 🚀 Key Features
- **Automated Scraping:** Extracts book titles, prices, ratings, and availability automatically.
- **Deep Scraping:** Navigates from listing pages to individual product detail pages to extract descriptions.
- **Pagination Logic:** Seamlessly moves through multiple pages using automated 'Next' button detection.
- **Data Cleaning:** Uses Pandas to clean price symbols and convert strings to numerical data for analysis.

## 📈 Visual Insights
The project includes data visualization to show price distributions and stock availability, helping business owners make data-driven pricing decisions.

## ⚙️ How to Run
1. Open the `.ipynb` file in Google Colab.
2. Run all cells.
3. The cleaned data will be exported as `books.csv`.
